# Проєкт: Тестування GreenCity Events

## 🔗 Сторінка для тестування
[GreenCity Events](https://www.greencity.cx.ua/#/greenCity/events)

## Що було зроблено:
- Створено структуру репозиторію.
- Проаналізовано функціонал сторінки.
- Написано 3 тест-кейси (включаючи негативний).

## ⚙️ Інструкція запуску
1. Склонуйте репозиторій:
   `git clone https://github.com/StelmakhNadiia/greencity-tests.git`
2. Встановіть залежності:
   `pip install -r requirements.txt`
3. Запустіть тести:
   `python -m unittest discover tests`
