# Тест-кейси для сторінки Events

### TC 1: Пошук події за назвою
**Preconditions:** Відкрита сторінка подій GreenCity.

| Step | Action | Data | Expected Result |
| :--- | :--- | :--- | :--- |
| 1 | Ввести назву існуючої події у поле пошуку | "Clean City" | У списку залишилися лише події з цією назвою |
| 2 | Очистити поле пошуку | - | Знову відображаються всі доступні події |

---

### TC 2: Фільтрація за типом події
**Preconditions:** Сторінка відкрита, є доступні події різних типів.

| Step | Action | Data | Expected Result |
| :--- | :--- | :--- | :--- |
| 1 | Натиснути на тег фільтра "Social" | "Social" | Відображаються лише події, що мають тег "Social" |

---

### TC 3 (Негативний): Пошук неіснуючої події
**Preconditions:** Сторінка відкрита.

| Step | Action | Data | Expected Result |
| :--- | :--- | :--- | :--- |
| 1 | Ввести випадковий набір символів у пошук | "qwerty12345" | Відображається повідомлення "Events not found" або порожня сторінка |
